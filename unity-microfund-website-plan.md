# Unity MicroFund Website Plan

## Executive Summary

Unity MicroFund is a Shariah-compliant micro-investment platform that enables collective investment in curated opportunities. This document outlines a complete website strategy covering sitemap, content structure, branding, and implementation guidelines for a professional, bilingual (English/Bangla) platform.

## 5. Investor Onboarding Flow

### Step-by-Step Journey

```
[1. Discovery]
    Landing on homepage → Browse content → View projects
        
[2. Interest]
    Click "Register" → Enter name, email, phone, password
        
[3. Verification]
    Email verification → KYC document upload (NID/Passport)
    → Admin verification (24-48 hours)
        
[4. Activation]
    Account approved → Email notification
        
[5. Funding]
    Browse projects → Select project → Choose amount
    → Payment gateway → Confirmation
        
[6. Investment]
    Receive investment certificate → Dashboard shows investment
    → Track progress
        
[7. Returns]
    Quarterly profit distribution → Notification → Dashboard update
    → Withdrawal or reinvest
```

### Registration Form Fields

| Field | Type | Required | Validation |
|-------|------|----------|------------|
| Full Name | Text | Yes | Min 2 chars |
| Email | Email | Yes | Valid email format |
| Phone | Tel | Yes | Valid BD format (+880) |
| NID Number | Text | Yes | 10 or 17 digits |
| Password | Password | Yes | Min 8 chars, 1 number |
| Referral Code | Text | No | Optional |
| Terms Acceptance | Checkbox | Yes | Must check |

### KYC Requirements

- Front of NID card (image)
- Back of NID card (image)
- Or Passport (for non-Bangladeshi)
- Proof of address (utility bill, if needed)

## 8. Brand Voice Guidelines

### Tone Attributes

| Attribute | Definition | Example |
|-----------|------------|---------|
| **Trustworthy** | Professional, honest, transparent | "We believe in complete transparency." |
| **Approachable** | Welcoming, easy to understand | "Investing made simple." |
| **Islamic Values** | References to faith, ethics, community | "Invest with Purpose. Earn with Faith." |
| **Professional** | Credible, experienced, reliable | "Shariah-compliant since 2024." |

### Voice Principles

1. **Always clarify** that investments carry risk
2. **Never promise specific returns** — use "projected" or "expected"
3. **Reference Shariah compliance** in every major section
4. **Use inclusive language** for all faiths while honoring Islamic principles
5. **Keep technical terms accessible** — define Islamic finance terms inline

## 10. Multi-Language Implementation (English + Bangla)

### Implementation Strategy

| Aspect | Recommendation |
|--------|---------------|
| **Architecture** | Single codebase with i18n library |
| **URL Structure** | `/en/` and `/bn/` subdirectories or query params |
| **Format** | `example.com/en/about` and `example.com/bn/about` |
| **Default** | English |
| **Persistence** | Cookie + localStorage for language selection |
| **Switcher** | Persistent dropdown in header |

### Library Recommendations

| Framework | Library |
|-----------|----------|
| Next.js | next-i18next, react-i18next |
| React (SPA) | react-i18next, i18next |
| Vue | vue-i18next |
| Static HTML | JSON-based custom implementation |

### Content Synchronization

| Element | Handling |
|---------|----------|
| **Body Text** | Full translation (EN ↔ BN) |
| **Headings** | Full translation |
| **Buttons** | Full translation |
| **Meta Tags** | Per-language SEO metadata |
| **URL Slugs** | `/projects` and `/bn/projects` |
| **Images** | Context-specific or stay same |

### Right-to-Left (RTL) Consideration

- English: Left-to-Right (LTR)
- Bangla: Left-to-Right (LTR)
- Note: No RTL needed as Bangla is LTR script

### Language Switcher UI

```html
<!-- Design Mockup -->
<div class="language-switcher">
  <button class="lang-btn">
    <span class="flag">🇬🇧</span> English
  </button>
  <span class="divider">|</span>
  <button class="lang-btn">
    <span class="flag">🇧🇩</span> বাংলা
  </button>
</div>
```

### Translation Workflow

1. **Content First**: Write all content in English
2. **Translation**: Professional translation to Bangla
3. **Review**: Native Bangla speaker review
4. **Integration**: Upload to translation management system

### SEO for Bilingual

| Element | Implementation |
|--------|----------------|
| **hreflang** | `<link rel="alternate" hreflang="en" href="..." />` |
| **hreflang** | `<link rel="alternate" hreflang="bn" href="..." />` |
| **Sitemap** | Separate entries per language URL |
| **Meta** | Separate meta descriptions per language |
